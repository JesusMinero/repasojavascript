Promesas y Asincronía en JavaScript

Manejo de operaciones asíncronas con callbacks, promesas y async/await

Introducción a la Asincronía

La asincronía en JavaScript permite ejecutar tareas sin bloquear el hilo principal del programa. Esto es útil cuando necesitas realizar operaciones que pueden llevar tiempo, como acceder a una API o leer archivos.

La asincronía se maneja comúnmente con callbacks, promesas y async/await.

Callbacks

Un callback en JavaScript es una función que se pasa como argumento a otra función y se ejecuta después de que se completa la tarea de la función que la recibió.

Código JavaScript

// Función con callback
function sumar(a, b, callback) {
  const resultado = a + b;
  callback(resultado);
}

// Callback que maneja el resultado
function mostrarResultado(resultado) {
  console.log("La suma es:", resultado);
}

// Llamada a la función con callback
sumar(5, 10, mostrarResultado);

Promesas

Una promesa en JavaScript es un objeto que representa la eventual finalización (o falla) de una operación asíncrona y su valor resultante.

Estructura básica de una promesa

// Crear una nueva promesa
const miPromesa = new Promise((resolve, reject) => {
  // Simular una operación asíncrona
  const exito = true; // Cambia a false para simular un error
  if (exito) {
    resolve("Operación exitosa!");
  } else {
    reject("Ocurrió un error.");
  }
});

// Usar la promesa
miPromesa
  .then((resultado) => {
    console.log(resultado);
  })
  .catch((error) => {
    console.error(error);
  });

Async/Await

async y await son palabras clave que permiten escribir código asincrónico de manera más clara y estructurada.

Estructura básica de async/await

// Función asíncrona
async function llamarApiConAsyncAwait() {
  try {
    let datos = await llamadaApi();
    console.log(datos);
  } catch (error) {
    console.log(error);
  }
}

// Llamar a la función
llamarApiConAsyncAwait();

Ejemplo Práctico: Obtener Datos de una API

Vamos a obtener datos de una API real (JSONPlaceholder) y mostrarlos en nuestra página.

Usando Promesas

function obtenerPosts() {
  return fetch('https://jsonplaceholder.typicode.com/posts')
    .then(response => {
      if (!response.ok) {
        throw new Error('Error en la llamada a la API');
      }
      return response.json();
    })
    .then(data => {
      mostrarPosts(data);
    })
    .catch(error => {
      console.log('Error:', error);
    });
}

Usando Async/Await

async function obtenerPostsConAsyncAwait() {
  try {
    let response = await fetch('https://jsonplaceholder.typicode.com/posts');
    if (!response.ok) {
      throw new Error('Error en la llamada a la API');
    }
    let posts = await response.json();
    mostrarPosts(posts);
  } catch (error) {
    console.log('Error:', error);
  }
}

Ejercicio Práctico: Comparación Promesas vs Async/Await

El objetivo de este ejercicio es crear una página web que obtenga una lista de usuarios desde una API y muestre sus nombres en una lista usando ambas estrategias.

Instrucciones:

  1. Crear la estructura HTML:
    • Dos botones: uno para cargar con promesas y otro con async/await
    • Un contenedor para mostrar la lista de usuarios
  2. Implementar ambas soluciones:
    • Usar fetch para obtener datos de https://jsonplaceholder.typicode.com/users
    • Implementar una solución con promesas (.then/.catch)
    • Implementar una solución con async/await
    • Mostrar los resultados en la página

Implementación:

Comparación: Promesas vs Async/Await

    Los usuarios aparecerán aquí

Solución:

// HTML
<button id="btnPromesas">Cargar con Promesas</button>
<button id="btnAsync">Cargar con Async/Await</button>
<ul id="user-list"></ul>

// JavaScript - Con Promesas
function cargarUsuariosConPromesas() {
  document.getElementById('user-list').innerHTML = '<li class="loading">Cargando con promesas...</li>';
  
  fetch('https://jsonplaceholder.typicode.com/users')
    .then(respuesta => {
      if (!respuesta.ok) throw new Error("Error en la respuesta");
      return respuesta.json();
    })
    .then(data => {
      mostrarUsuarios(data, "Con Promesas");
    })
    .catch(error => {
      document.getElementById('user-list').innerHTML = `<li class="error">Error: ${error.message}</li>`;
    });
}

// JavaScript - Con Async/Await
async function cargarUsuariosConAsync() {
  document.getElementById('user-list').innerHTML = '<li class="loading">Cargando con async/await...</li>';
  
  try {
    const respuesta = await fetch('https://jsonplaceholder.typicode.com/users');
    if (!respuesta.ok) throw new Error("Error en la respuesta");
    const data = await respuesta.json();
    mostrarUsuarios(data, "Con Async/Await");
  } catch (error) {
    document.getElementById('user-list').innerHTML = `<li class="error">Error: ${error.message}</li>`;
  }
}

// Función para mostrar usuarios
function mostrarUsuarios(usuarios, metodo) {
  const lista = document.getElementById('user-list');
  lista.innerHTML = `<li><strong>${metodo}</strong></li>`;
  
  usuarios.forEach(usuario => {
    const li = document.createElement("li");
    li.className = "user-item";
    li.textContent = `${usuario.name} - ${usuario.email}`;
    lista.appendChild(li);
  });
}

// Event listeners
document.getElementById('btnPromesas').addEventListener('click', cargarUsuariosConPromesas);
document.getElementById('btnAsync').addEventListener('click', cargarUsuariosConAsync);
