Manipulación del DOM con JavaScript

Ejercicios básicos de manipulación del DOM con JavaScript

Introducción

El DOM (Document Object Model) es una representación en forma de árbol de los elementos de un documento HTML. JavaScript nos permite manipular este DOM para cambiar el contenido, la estructura y el estilo de una página web de forma dinámica.

Selección de Elementos del DOM

En este ejemplo, seleccionaremos elementos del DOM usando diferentes métodos.

Métodos de Selección

// Seleccionar por ID
const elementById = document.getElementById('myId');

// Seleccionar por clase (devuelve HTMLCollection)
const elementsByClass = document.getElementsByClassName('myClass');

// Seleccionar por etiqueta (devuelve HTMLCollection)
const elementsByTag = document.getElementsByTagName('p');

// Seleccionar con querySelector (primer elemento que coincide)
const elementQuery = document.querySelector('.myClass');

// Seleccionar con querySelectorAll (todos los elementos que coinciden)
const elementsQueryAll = document.querySelectorAll('p.myClass');

Manipulación de Contenido y Estilos

Aquí cambiaremos el texto y los estilos de un elemento del DOM.

Código JavaScript

const myDiv = document.getElementById('myDiv');

// Cambiar el texto del elemento
document.getElementById('changeText').addEventListener('click', () => {
  myDiv.textContent = 'Text Changed';
});

// Cambiar el estilo del elemento
document.getElementById('changeStyle').addEventListener('click', () => {
  myDiv.style.color = 'red';
  myDiv.style.fontSize = '24px';
});

Hello, World!

Eventos del DOM

En este ejemplo, manejaremos eventos del DOM para interactuar con los elementos de la página.

Código JavaScript

const myDiv = document.getElementById('myDiv');

// Manejar el evento click
myDiv.addEventListener('click', () => {
  alert('Div clicked!');
  myDiv.style.backgroundColor = 'lightblue';
});

// Manejar el evento mouseover
myDiv.addEventListener('mouseover', () => {
  myDiv.style.backgroundColor = 'yellow';
});

// Manejar el evento mouseout
myDiv.addEventListener('mouseout', () => {
  myDiv.style.backgroundColor = '';
});

Resultado

¡Pasa el mouse sobre mí o haz clic!

Manipulación Completa del DOM

Crea una página HTML con varios elementos y utiliza JavaScript para realizar diversas manipulaciones.

Código JavaScript

// Cambiar el contenido del título
document.getElementById('changeContent').addEventListener('click', () => {
  document.getElementById('title').textContent = 'Content Changed!';
});

// Resaltar todos los párrafos
document.getElementById('highlightText').addEventListener('click', () => {
  const paragraphs = document.querySelectorAll('.text');
  paragraphs.forEach(p => {
    p.classList.toggle('highlight');
  });
});

Ejercicio Práctico

Crear una página HTML simple y usar JavaScript para seleccionar elementos del DOM, manipular su contenido y estilos, y manejar eventos del DOM.

Instrucciones:

  1. Crear una página HTML básica:
    • Incluir un encabezado (h1), un párrafo (p), una lista desordenada (ul) con varios elementos de lista (li) y un botón (button).
  2. Usar JavaScript para manipular el DOM:
    • Seleccionar elementos del DOM usando getElementById, getElementsByClassName, getElementsByTagName, querySelector, y querySelectorAll.
    • Cambiar el contenido del encabezado y del párrafo.
    • Cambiar los estilos de los elementos de lista.
    • Agregar y quitar clases CSS a los elementos.
  3. Manejar eventos del DOM:
    • Agregar un evento click al botón para cambiar el contenido del encabezado.
    • Agregar eventos mouseover y mouseout a los elementos de la lista para cambiar su color de fondo cuando el mouse pasa sobre ellos.

Solución:

<!-- HTML -->
<h1 id="mainTitle">Título Principal</h1>
<p class="description">Este es un párrafo de ejemplo.</p>
<ul id="myList">
  <li>Elemento 1</li>
  <li>Elemento 2</li>
  <li>Elemento 3</li>
</ul>
<button id="changeTitle">Cambiar Título</button>

// JavaScript
// Seleccionar elementos
const title = document.getElementById('mainTitle');
const description = document.querySelector('.description');
const listItems = document.querySelectorAll('#myList li');
const changeButton = document.getElementById('changeTitle');

// Cambiar contenido
changeButton.addEventListener('click', () => {
  title.textContent = '¡Título Cambiado!';
  description.textContent = 'El contenido ha sido modificado.';
});

// Cambiar estilos de lista
listItems.forEach(item => {
  item.style.color = 'blue';
});

// Eventos mouseover y mouseout
listItems.forEach(item => {
  item.addEventListener('mouseover', () => {
    item.style.backgroundColor = 'lightgray';
  });
  item.addEventListener('mouseout', () => {
    item.style.backgroundColor = '';
  });
});
