Manipulacion del DOM

DOM

El DOM (Document Object Model) es una representación en forma de árbol de los elementos de un documento HTML. JavaScript nos permite manipular este DOM para cambiar el contenido, la estructura y el estilo de una página web de forma dinámica. Domínalas con estos ejercicios prácticos.

  • Tipos de acceso
  • Eventos
