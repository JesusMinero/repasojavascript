Iteración sobre Arreglos

Una técnica fundamental en el manejo de arreglos es la iteración. Veamos diferentes formas de iterar sobre arreglos:

Bucle for tradicional

const frutas = ["manzana", "banana", "naranja", "uva"];
for (let i = 0; i < frutas.length; i++) {
  console.log(frutas[i]);

Métodos de Arreglos

Método map()

El método map crea un nuevo arreglo con los resultados de llamar a una función para cada elemento.

const numeros = [1, 2, 3, 4, 5];
const numerosDuplicados = numeros.map(num => num * 2);
console.log(numerosDuplicados); // [2, 4, 6, 8, 10]

Resultado:

Método filter()

Filter crea un nuevo arreglo con todos los elementos que pasan la prueba implementada por la función proporcionada.

const numeros = [1, 2, 3, 4, 5];
const numerosFiltrados = numeros.filter(num => num > 2);
console.log(numerosFiltrados); // [3, 4, 5]

Resultado:

Método reduce()

Reduce ejecuta una función reductora sobre cada elemento de un array, devolviendo como resultado un único valor.

const numeros = [1, 2, 3, 4, 5];
const suma = numeros.reduce((total, num) => total + num, 0);
console.log(suma); // 15

Objetos en JavaScript

Creación y Acceso a Propiedades

Los objetos en JavaScript son estructuras que permiten almacenar datos en pares clave-valor.

const persona = {
  nombre: "Juan",
  edad: 30,
  ciudad: "Madrid"
};

// Notación de punto
console.log(persona.nombre); // Juan

// Notación de corchetes
console.log(persona["edad"]); // 30

Resultado:

Iteración sobre Propiedades

Para recorrer las propiedades de un objeto, podemos usar un bucle for...in:

const libro = {
  titulo: "El principito",
  autor: "Antoine de Saint-Exupéry",
  año: 1943
};

for (let propiedad in libro) {
  console.log(`${propiedad}: ${libro[propiedad]}`);
}

Transformación de Arreglos de Objetos

Extraer una propiedad específica

Podemos usar map() para extraer una propiedad específica de cada objeto en un arreglo.

const usuarios = [
  {nombre: "Ana", edad: 25},
  {nombre: "Luis", edad: 30},
  {nombre: "María", edad: 28}
];

function obtenerNombres(arr) {
  return arr.map(usuario => usuario.nombre);
}

console.log(obtenerNombres(usuarios));
// ["Ana", "Luis", "María"]

Resultado:

Filtrado de Arreglos de Objetos

Podemos usar filter() para crear un nuevo arreglo con objetos que cumplan cierta condición.

function filtrarUsuarios(arr) {
  return arr.filter(usuario => usuario.edad > 27);
}

console.log(filtrarUsuarios(usuarios));
// [{ nombre: "Luis", edad: 30 }, { nombre: "María", edad: 28 }]

Resultado:

Agrupación por Propiedad

Podemos usar reduce() para agrupar objetos en un arreglo por una propiedad específica.

function agruparPorCategoria(arr) {
  return arr.reduce((grupo, producto) => {
    const categoria = producto.categoria;
    if (!grupo[categoria]) {
      grupo[categoria] = [];
    }
    grupo[categoria].push(producto);
    return grupo;
  }, {});
}

const productos = [
  {nombre: "camiseta", categoria: "ropa"},
  {nombre: "pantalón", categoria: "ropa"},
  {nombre: "zapatos", categoria: "calzado"},
  {nombre: "sandalias", categoria: "calzado"}
];

console.log(agruparPorCategoria(productos));

Ejercicio 1: Operaciones con Arreglos

Investiga o crea una función que utilice map, filter y reduce para procesar un arreglo de números:

  • Duplicar los números pares (map)
  • Filtrar los números mayores a 10 (filter)
  • Sumar los números resultantes (reduce)
const numeros = [5, 12, 8, 21, 4, 15];

function procesarNumeros(arr) {
  // Tu código aquí
}

console.log(procesarNumeros(numeros)); // Resultado esperado: 52
function procesarNumeros(arr) {
  return arr
    .map(num => num % 2 === 0 ? num * 2 : num)
    .filter(num => num > 10)
    .reduce((total, num) => total + num, 0);
}
